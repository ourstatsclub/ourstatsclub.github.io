# R Relational Operators

| Relational Operators in R |  |
| :--- | :--- |
| Operator | Description |
| \&lt; | Less than |
| \&gt; | Greater than |
| \&lt;= | Less than or equal to |
| \&gt;= | Greater than or equal to |
| == | Equal to |
| != | Not equal to |

Some examples are:

`x = 5`

`y = 16`

`x\<y`

`[1] TRUE`

`x\>y`

`[1] FALSE`

`x\<=5`

`[1] TRUE`

`y\>=20`

`[1] FALSE`

`y == 16`

`[1] TRUE`

`x != 5`

`[1] FALSE`
